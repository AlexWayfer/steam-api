# SteamAPI

`steam-api` is a simple gem to expose the Steam WebApi directly.

## Installation

Add this line to your application's Gemfile:

    gem 'simple-steam'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install simple-steam
